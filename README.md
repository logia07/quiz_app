# 🎀 Анкета-генератор для конкурса

Мини-приложение для VK и Telegram: пользователь проходит анкету → получает персонализированную иллюстрацию → делится ею в соцсетях.  
Идеально для виральных конкурсов среди девушек 💕

---

## ✨ Возможности
- Ввод данных на 8 экранах в виде анкеты для девочек
- Мгновенная генерация красивой иллюстрации (1080×4830 px)
- Автоотправка результата в ЛС Telegram
- Публикация на стене VK через кнопку «Поделиться»
- Поддержка кириллицы и элегантного шрифта `Blogger_Sans-Light_Italic`
